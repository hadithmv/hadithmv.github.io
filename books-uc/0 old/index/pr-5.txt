You've made an excellent point, and your diagnosis is spot on. The dark mode colors were not well-calibrated, leading to poor contrast, especially with the `--primary` color being too dark for text. The search bar was also too similar to the background.

I have created a completely revised, professionally designed dark mode palette. This new palette is built on principles of accessibility and visual hierarchy, ensuring that text is always easy to read and important elements stand out.

### Summary of Fixes:

1.  **New Dark Mode Palette:** I've replaced the old dark mode variables with a more sophisticated set of colors. The new background is a soft dark blue-grey, with lighter shades for cards and the header to create a sense of depth.
2.  **High-Contrast Text:** Section titles and book titles now use a bright, almost white color in dark mode, making them pop against the darker backgrounds.
3.  **Visible Search Bar:** The search bar now has a lighter background and a subtle border in dark mode, making it clearly visible and defined.
4.  **Cohesive Header:** The header color is now a lighter shade of the background, distinguishing it clearly from the main content area.
